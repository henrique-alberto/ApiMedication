# ApiMedication

ASP.NET Core WEB API project developed in C#.

This project uses:

- SQL Server database
- .NET 6.0
- Entity Framework Core
- xUnit.

In this project you can:

- Add medication 
- List medications
- Find by Id
- Delete medication
